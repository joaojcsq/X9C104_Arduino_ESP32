 Programa para o CI x9c104.
 
 O x9c104 tem 100kOhms (10^4) e resolução de 1K por pulso (0 até 99).
 
 O programa inicia zerando o CI (posição 0).
 
 Comandos para usar na Serial:
                         
                           a --> Aumenta a posição
                           d --> Diminui a posição
                           z --> Zera a posição
                           m --> Salva a posição na memória do CI
